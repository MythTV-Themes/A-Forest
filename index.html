Installation

  • Theme Chooser is not up to date, until it will be you can get it with git:

cd ~/.mythtv/themes/
git clone -b mythtv-master git://github.com/MythTV-Themes/A-Forest.git

  • If you want to get the latest developing version, you can get it with subversion:

cd ~/.mythtv/themes/
git clone git://github.com/MythTV-Themes/A-Forest.git

For get the update

cd ~/.mythtv/themes/A-Forest
git pull
